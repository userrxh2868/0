Termux 终极配置教程

欢迎来到最详细的 Termux 手机编程环境配置指南。本教程采用 Material Design 3 风格设计，专为小白打造，手把手教你将手机变成强大的移动开发终端。

第一步：下载与初始化

download_for_offline 1. 正确下载 Termux

警告 不要使用 Google Play 版本的 Termux，那个版本已经停止更新很久了，会有各种报错。

推荐 请下载 F-Droid 版本的 Termux。这是官方维护的最新版本。

open_in_new 前往 F-Droid 下载页面
update 2. 第一次进入的仪式

安装好后打开 App，你需要做两件事：更新软件源和获取存储权限。请依次执行以下命令（点击复制）：

BASH
# 1. 更新所有软件包 (遇到问号一直按回车即可) pkg update && pkg upgrade -y # 2. 获取手机存储访问权限 (会弹出弹窗，点允许) termux-setup-storage

第二步：网络加速 (免费VPN)

rocket_launch 为什么需要加速？

Termux 安装软件是从国外的服务器下载的，如果没有加速器，速度可能会只有 5kb/s，甚至安装失败。

这里推荐两款在 Google Play 上免费、良心、且对新手友好的 VPN/加速工具。安装后开启，再回到 Termux 输入命令，速度会飞起。

1. Cloudflare WARP (1.1.1.1)

全球最良心的免费 DNS 加速器，不限流量，一键开启。它不是传统梯子，但能极大加速 GitHub 和 Termux 源的连接。

Google Play 搜索 "1.1.1.1"

2. Proton VPN

瑞士大厂，提供免费且无广告的无限流量节点（通常是荷兰、美国、日本）。速度很快，适合下载大文件。

Google Play 搜索 "Proton VPN"
sync_alt 进阶：更换清华源 (无需VPN)

如果你不想安装 VPN，可以通过命令将下载源切换到国内的清华大学镜像源。

BASH
termux-change-repo # 执行后会出现图形界面： # 1. 使用方向键选择所有方块 (按空格选中) -> 回车 # 2. 选择 Mirrors by Tsinghua -> 回车

第三步：安装编程语言

code Python (爬虫、脚本之王)

Python 是 Termux 上最热门的语言，可以用来跑脚本、爬虫、甚至搭建简单的 Web 服务器。

安装指令
# 安装 Python pkg install python -y # 验证安装 (查看版本) python --version # 升级 pip (包管理器) pip install --upgrade pip
Hello World 测试
# 直接在命令行输入 python 进入交互模式，然后输入： print("Hello Termux MD3!") # 按 Ctrl+D 退出
javascript Node.js (前端、服务端)

如果你想在手机上运行 Hexo 博客、Vue 项目或者各种 JS 脚本，Node.js 是必须的。这里安装的是 LTS (长期支持) 版本。

安装指令
# 安装 Node.js LTS 版本 pkg install nodejs-lts -y # 或者安装最新版: pkg install nodejs -y # 验证安装 node -v npm -v
coffee Java (JDK 17)

Java 是安卓开发的基石。Termux 提供了 OpenJDK 17，非常适合学习 Java 基础或运行 `.jar` 工具。

安装指令
# 安装 OpenJDK 17 pkg install openjdk-17 -y # 验证安装 java --version # 编译 Java 文件命令: javac Hello.java # 运行 Java 文件命令: java Hello
memory C / C++ (Clang 编译器)

底层开发必备。在 Termux 中，我们使用 `clang` 来编译 C/C++ 代码，它比 GCC 更轻量且报错信息更友好。

安装指令
# 安装 Clang 和 Make 工具 pkg install clang make -y # 验证安装 clang --version clang++ --version # 编译 C++ 示例: clang++ main.cpp -o main # 运行: ./main
go_to_line Go Lang (高并发)

Google 开发的语言，非常适合编写网络服务。Termux 对 Go 的支持非常完美。

安装指令
# 安装 Go pkg install golang -y # 验证安装 go version # 建议配置 GOPROXY 加速包下载 go env -w GOPROXY=https://goproxy.cn,direct
settings_suggest Rust (现代、安全)

近年来最火的系统级语言，虽然安装包较大，但 Termux 依然能够运行。安装过程可能会比较慢，请保持 VPN 开启。

安装指令
# 安装 Rust 编译器 pkg install rust -y # 验证安装 rustc --version cargo --version
html PHP & Web Server

如果你想运行简单的网站或者学习 PHP 后端。同时推荐安装 nginx 作为服务器。

安装指令
# 安装 PHP pkg install php -y # 安装 Nginx (可选) pkg install nginx -y # 启动 PHP 内置服务器 (在当前目录) php -S 0.0.0.0:8080

第四步：神器级工具

handyman Git, Vim & Zsh

只有语言还不够，你需要代码管理工具 (Git)、编辑器 (Vim/Nano) 和更漂亮的终端 (Zsh)。

一键安装全家桶
# 安装 Git (版本控制) pkg install git -y # 安装 Vim (编辑器之神) 和 Nano (新手友好) pkg install vim nano -y # 安装 wget 和 curl (下载工具) pkg install wget curl -y # 安装 Tree (查看目录结构) pkg install tree -y
auto_fix_high 终极美化：Oh My Zsh

让你的 Termux 界面变得像黑客电影一样酷炫，带有自动补全和主题功能。

BASH (需要开启VPN)
# 这是一个一键安装脚本 (来自 tmoe) sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
