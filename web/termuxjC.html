什么是Termux？

Termux 是一个Android终端模拟器和Linux环境应用程序，无需root权限即可运行。

它提供了一个完整的Linux环境，你可以在手机上安装各种编程语言和工具，实现真正的移动开发。

为什么选择Termux？

  • 完全免费 - 开源项目，无任何费用
  • 无需root - 普通Android手机即可使用
  • 完整Linux环境 - 支持apt包管理
  • 多语言支持 - 可安装10+种编程语言
  • 便携性 - 随时随地进行开发

初始配置

1. 基础更新

打开Termux，执行以下命令：

pkg update -y && pkg upgrade -y

这个命令会更新包列表和升级已安装的包

2. 设置存储权限

termux-setup-storage

授予文件访问权限，这样Termux可以访问手机存储

3. 更换国内镜像源（加速）

国内用户强烈建议更换镜像源以加速下载：

termux-change-repo

按照提示选择：
1. 选择 Mirrors by 清华大学
2. 按回车确认

安装编程语言

以下命令可以一次性安装所有语言，也可以单独安装。

Go Python Node.js Java Rust C++ PHP Ruby Perl Lua

一键安装所有语言

# 一次性安装所有编程语言（约1.5GB）
pkg install golang python nodejs openjdk-17 \
            rust clang php ruby perl lua53 \
            make cmake git vim -y

预计时间：10-30分钟（取决于网速）

所需空间：约1.5GB

单独安装（按需选择）

# Python（最通用）
pkg install python -y

# Node.js（JavaScript）
pkg install nodejs -y

# Go（高性能后端）
pkg install golang -y
go env -w GOPROXY=https://goproxy.cn,direct

# C/C++（系统编程）
pkg install clang make -y

# Java（Android开发基础）
pkg install openjdk-17 -y

VPN使用技巧

  • 下载时开启：安装软件包时开VPN，速度更快
  • 编程时关闭：日常写代码可关VPN省电
  • 多节点测试：不同节点速度差异大，多试试
  • 注意电量：VPN会增加耗电，注意手机电量

项目实战

1. 创建第一个Python项目

# 创建项目目录
mkdir -p ~/projects/myapp && cd ~/projects/myapp

# 创建Python文件
cat > app.py << 'EOF'
#!/usr/bin/env python3
import os
import time

print("我的第一个Termux应用！")
print(f"当前目录: {os.getcwd()}")
print(f"当前时间: {time.strftime('%Y-%m-%d %H:%M:%S')}")

# 简单计算
numbers = [1, 2, 3, 4, 5]
total = sum(numbers)
print(f"列表{numbers}的和是: {total}")
EOF

# 运行程序
python app.py

2. 创建简单的HTTP服务器

# Python简单HTTP服务器
cd ~/storage/shared  # 进入手机共享目录
python -m http.server 8080

# 访问：用浏览器打开
# 手机：http://localhost:8080
# 电脑：http://手机IP:8080

使用技巧

提高效率的技巧

  • 命令历史：按音量上+键盘上箭头，查看历史命令
  • Tab补全：输入命令时按Tab键自动补全
  • 长按复制：在Termux中长按可以复制文本
  • 分屏操作：长按Termux图标，选择"浮动窗口"

省电技巧

  • 关闭不必要的后台进程
  • 使用深色主题（OLED屏幕省电）
  • 不需要时关闭VPN
  • 适当降低屏幕亮度

常见问题

Q1: 安装时提示"空间不足"怎么办？

A: 清理手机存储空间，或运行：

pkg clean  # 清理包缓存

Q2: 网络慢，下载失败怎么办？

A:
1. 换到WiFi环境下操作
2. 使用VPN加速
3. 更换国内镜像源
4. 分多次安装，不要一次性装太多

Q3: 如何备份Termux环境？

# 备份到手机存储
cd ~
tar -zcvf /sdcard/termux-backup.tar.gz .
