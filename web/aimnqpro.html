逻辑规范文档

1. 字典模式 (Dictionary)

AI 根据用户输入的关键词回复。数据格式为 JSON 对象。

示例:
{ "你好": "你好呀！", "名字": "我是 MD3 AI 助手。", "default": "我不明白你的意思。" }

注：如果找不到匹配的词，会回复 "default" 的内容。

2. 规则模式 (Sequence)

AI 将无视用户的具体内容，严格按照列表顺序回复。数据格式为 JSON 数组。

示例:
[ "第一步：请点击开始", "第二步：输入你的账号", "第三步：流程结束，再见" ]

注：当数组内容说完后，会循环回到第一句（或停止，视逻辑而定）。
